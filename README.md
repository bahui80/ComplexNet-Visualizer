# Visualizador de topologías

Proyecto de visualización de topologías realizado en Unity3d versión 5.3.5f1.

## Modificaciones al visualizador

Descargarse Unity3d desde [Unity](https://store.unity.com/es/download?ref=personal). El proyecto está compilado bajo la versión 5.3.5f1, por lo tanto debe descargarse esta misma o una versión posterior.

Al instalar Unity3d se preguntará qué compiladores se quiere instalar. Estos servirán luego para generar el proyecto en las distintas plataformas (Linux, Mac, Windows, Android, etc).

Para modificar el código, importar el proyecto en Unity3d haciendo doble click en la escena ubicada en Assets/Scenes/Topology.unity. En caso de no haberse cargado la escena, desde Unity3d hacer doble click nuevamente en la escena Topology.unity.

## Compilación del visualizador
Para compilar el visualizador basta con ir a File -> Build Settings y allí elegir con qué plataforma se quiere compilar el visualizador.
