# Visualización de topologías

Proyecto de visualización de topologías realizado en Unity3d versión 5.3.5f1.

## Modificaciones al visualizado

Para modificar el código, importar el proyecto en Unity haciendo click en la escena ubicada en /Assets/Scenes/Topology.unity

## Como correr el visualizador

### Linux

Para ejecutar el visualizador ejecutar el archivo Topology.x86_64

### Windows

Para ejecutar el visualizador ejecutar el archivo Topology.exe

### Mac

Para ejecutar el visualizador ejecutar el archivo Topology.app
